# 游戏设计

- 通过副本获得装备与金币
- 装备分为五个等级：破旧、普通、神器、史诗、独特
- 主角默认每秒回血 2%
- 默认暴击伤害 150%
- 完成副本时可能掉落独特级别(5 个词条)的装备（通关后 2.5%掉落率，无尽挑战不会掉落）
- 通关后开启无尽挑战

# 本地运行
